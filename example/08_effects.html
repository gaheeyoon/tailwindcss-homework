👁️ Opacity — 투명도

📖 Opacity 이론

Opacity는 요소의 투명도를 조절합니다. 0(완전 투명)부터 100(불투명)까지 설정할 수 있습니다.

기본 값:

  • • opacity-0 → 완전 투명
  • • opacity-25 → 25% 불투명
  • • opacity-50 → 50% 불투명
  • • opacity-75 → 75% 불투명
  • • opacity-100 → 완전 불투명

임의 값:

  • • opacity-[0.35] → 35%
  • • opacity-[0.65] → 65%
  • • 소수점으로 세밀한 조절 가능

👆 Cursor — 커서 스타일

📖 Cursor 이론

Cursor는 마우스 커서의 모양을 변경합니다.

자주 쓰는 커서:

  • • cursor-default → 기본 화살표
  • • cursor-pointer → 손가락 포인터
  • • cursor-not-allowed → 금지 표시
  • • cursor-wait → 로딩

기타 커서:

  • • cursor-help → 도움말
  • • cursor-text → 텍스트 선택
  • • cursor-move → 이동
  • • cursor-grab → 잡기

🌫️ Shadow — 그림자

📖 Shadow 이론

Shadow는 요소에 그림자 효과를 추가합니다.

그림자 크기:

  • • shadow-sm → 아주 작은 그림자
  • • shadow → 기본 그림자
  • • shadow-md → 중간 그림자
  • • shadow-lg → 큰 그림자
  • • shadow-xl → 아주 큰 그림자
  • • shadow-2xl → 초대형 그림자

그림자 없음:

  • • shadow-none → 그림자 제거

색상 그림자:

  • • shadow-blue-500/50 → 파란색 그림자
  • • shadow-red-500/30 → 빨간색 그림자

🖱️ Hover — 마우스 올렸을 때

📖 Hover 이론

Hover는 마우스를 요소 위에 올렸을 때 적용되는 스타일입니다.

색상 변경:

  • • hover:bg-blue-600 → 배경색
  • • hover:text-red-500 → 텍스트색
  • • hover:border-green-500 → 테두리색

효과 변경:

  • • hover:shadow-lg → 그림자
  • • hover:opacity-80 → 투명도
  • • hover:scale-105 → 크기

Transform:

  • • hover:scale-110 → 확대
  • • hover:-translate-y-1 → 위로
  • • hover:rotate-6 → 회전

Transition (부드럽게):

  • • transition → 기본 전환
  • • duration-300 → 0.3초

Active는 요소를 클릭(마우스를 누르고 있는)할 때 적용되는 스타일입니다.

  • • 클릭했다는 시각적 피드백

👥 Group — 부모 hover로 자식 제어

📖 Group 이론

Group은 부모 요소에 마우스를 올렸을 때 자식 요소를 변경하는 강력한 기능입니다.

사용 방법:

  • • 부모에 group 클래스
  • • 자식에 group-hover: 접두사
  • • 예: group-hover:text-blue-500

Tip:

  • • 카드 hover 시 내부 요소 변경
  • • 버튼 hover 시 아이콘 움직임
  • • 링크 hover 시 화살표 이동

실습 18: Group 기본 사용

📝 실습 과제

  • • 카드에 group 클래스
  • • 제목: group-hover 시 파란색
  • • 설명: group-hover 시 회색→검정

🔗 Peer — 형제 요소 제어

📖 Peer 이론

Peer는 형제 요소의 상태에 따라 다른 요소를 변경합니다. 주로 체크박스, 라디오 버튼과 함께 사용합니다.

사용 방법:

  • • 기준 요소에 peer 클래스
  • • 대상 요소에 peer-*: 접두사
  • • 예: peer-checked:bg-blue-500

Tip:

  • • 체크박스 선택 시 레이블 스타일
  • • 라디오 버튼 커스텀 디자인
  • • 토글 스위치 구현

실습 20: Peer 체크박스

  • • input에 peer 클래스
