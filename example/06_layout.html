📐 Flexbox — 1차원 레이아웃

📖 Flexbox 이론

Flexbox는 가로 또는 세로 한 방향으로 요소를 배치하는 레이아웃입니다.

기본 설정:

  • • flex → 가로 배치
  • • flex-col → 세로 배치
  • • flex-wrap → 줄바꿈 허용
  • • flex-nowrap → 한 줄 유지

가로축 정렬 (justify):

  • • justify-start → 왼쪽
  • • justify-center → 가운데
  • • justify-end → 오른쪽
  • • justify-between → 양 끝 + 사이 균등
  • • justify-around → 균등 간격

세로축 정렬 (items):

  • • items-start → 위쪽
  • • items-center → 가운데
  • • items-end → 아래쪽
  • • items-stretch → 높이 맞춤

간격 & 크기:

  • • gap-4 → 요소 간격 1rem
  • • flex-1 → 남은 공간 채우기
  • • flex-grow → 늘어나기
  • • flex-shrink-0 → 줄어들지 않음

🎯 Grid — 2차원 레이아웃

📖 Grid 이론

Grid는 행(row)과 열(column)을 동시에 다루는 2차원 레이아웃입니다.

기본 설정:

  • • grid → Grid 선언
  • • grid-cols-3 → 3개 열
  • • grid-rows-2 → 2개 행
  • • gap-4 → 간격 1rem

열 병합 (col-span):

  • • col-span-2 → 2칸 차지
  • • col-span-3 → 3칸 차지
  • • col-span-full → 전체 차지

행 병합 (row-span):

  • • row-span-2 → 2행 차지
  • • row-span-3 → 3행 차지

정렬:

  • • place-items-center → 아이템 중앙
  • • place-content-center → 전체 중앙
  • • justify-items-start → 가로 정렬
  • • items-start → 세로 정렬

📱 반응형 Grid

📖 반응형 Grid 이론

반응형 Grid는 화면 크기에 따라 열 개수를 자동으로 조정합니다.

브레이크포인트:

  • • sm: → 640px 이상
  • • md: → 768px 이상
  • • lg: → 1024px 이상
  • • xl: → 1280px 이상

사용 예시:

  • • grid-cols-1 md:grid-cols-3
  • • 모바일: 1열, 태블릿 이상: 3열
