📱 Responsive 기본 개념

📖 Responsive 이론

Tailwind는 모바일 퍼스트(Mobile-first) 방식입니다. 기본 클래스는 모바일에 적용되고, 브레이크포인트를 사용해 큰 화면에서의 스타일을 추가합니다.

브레이크포인트:

  • • sm: → 640px 이상
  • • md: → 768px 이상
  • • lg: → 1024px 이상
  • • xl: → 1280px 이상
  • • 2xl: → 1536px 이상

숨김/보임:

  • • hidden → 숨김
  • • block → 블록으로 표시
  • • hidden md:block → md 이상에서 보임
  • • block md:hidden → md 이상에서 숨김

자주 쓰는 클래스:

  • • container → 반응형 컨테이너
  • • mx-auto → 가로 중앙 정렬
  • • max-w-7xl → 최대 너비 제한
  • • aspect-video → 16:9 비율

레이아웃 전환:

  • • flex-col md:flex-row → 방향 전환
  • • w-full md:w-1/2 → 너비 전환
  • • text-sm md:text-lg → 크기 전환

📖 반응형 Grid 이론

Grid는 화면 크기에 따라 열 개수를 자동으로 조정할 수 있습니다.

기본 패턴:

  • • grid-cols-1 → 모바일 1열
  • • sm:grid-cols-2 → sm 이상 2열
  • • md:grid-cols-3 → md 이상 3열
  • • lg:grid-cols-4 → lg 이상 4열

예시:

  • • 상품 목록: 1열 → 2열 → 3열
  • • 이미지 갤러리: 2열 → 3열 → 4열
  • • 대시보드: 1열 → 2열 → 3열

📖 자주 쓰는 유틸리티 이론

자주 사용하는 컨테이너, 최대 너비, 비율 관련 유틸리티입니다.

Container & 중앙 정렬:

  • • container → 반응형 컨테이너
  • • mx-auto → 가로 중앙 정렬
  • • px-4 → 좌우 여백

최대 너비 (max-w):

  • • max-w-md → 28rem
  • • max-w-lg → 32rem
  • • max-w-2xl → 42rem
  • • max-w-7xl → 80rem

비율 (aspect):

  • • aspect-square → 1:1
  • • aspect-video → 16:9
  • • aspect-[4/3] → 4:3

사용 예시:

  • • 페이지 레이아웃: container + mx-auto
  • • 콘텐츠 영역: max-w-4xl + mx-auto
  • • 이미지/비디오: aspect-video
