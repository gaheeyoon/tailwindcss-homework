📦 Padding — 요소 안쪽 여백

📖 Padding 이론

Padding은 요소의 내부 여백으로, 콘텐츠와 테두리 사이의 공간을 의미합니다.

전체 방향:

  • • p-4 → 상하좌우 모두 1rem (16px)
  • • p-8 → 상하좌우 모두 2rem (32px)

축 방향:

  • • px-6 → 좌우(x축) 1.5rem
  • • py-8 → 상하(y축) 2rem

개별 방향:

  • • pt-2 → 위쪽(top) 0.5rem
  • • pr-4 → 오른쪽(right) 1rem
  • • pb-6 → 아래쪽(bottom) 1.5rem
  • • pl-8 → 왼쪽(left) 2rem

숫자 기준:

  • • 1 = 0.25rem (4px)
  • • 4 = 1rem (16px)
  • • 8 = 2rem (32px)

📦 Margin — 요소 바깥 여백

📖 Margin 이론

Margin은 요소의 외부 여백으로, 다른 요소와의 간격을 의미합니다.

기본 사용법:

  • • m-4 → 상하좌우 모두 1rem
  • • mx-6 → 좌우 1.5rem
  • • my-8 → 상하 2rem

특별한 값:

  • • mx-auto → 가로 중앙 정렬
  • • -mt-4 → 음수 margin (위로 당기기)
  • • -ml-8 → 음수 margin (왼쪽으로 당기기)

개별 방향:

  • • mt-4 → 위쪽 margin
  • • mr-6 → 오른쪽 margin
  • • mb-8 → 아래쪽 margin
  • • ml-2 → 왼쪽 margin

주의사항:

  • • Margin은 다른 요소와의 간격
  • • 음수 margin으로 겹치기 가능
  • • mx-auto는 블록 요소 중앙 정렬

📦 Space & Gap — 요소 간 간격

📖 Space Between & Gap 이론

Space Between과 Gap은 여러 자식 요소들 사이의 간격을 한 번에 설정하는 유틸리티입니다.

Space Between:

  • • space-y-4 → 세로 간격 1rem
  • • space-x-6 → 가로 간격 1.5rem
  • • 부모 요소에 적용하면 자식들 사이에 margin 생성

Gap (Flex/Grid용):

  • • gap-4 → 모든 방향 1rem
  • • gap-x-6 → 가로 간격만 1.5rem
  • • gap-y-8 → 세로 간격만 2rem
  • • Flexbox나 Grid에서 사용

📦 Width / Height — 크기 설정

📖 Width / Height 이론

Width와 Height는 요소의 크기를 설정합니다.

단계 기반:

  • • w-10 → 2.5rem (40px)
  • • w-20 → 5rem (80px)
  • • h-10 → 2.5rem (40px)
  • • h-20 → 5rem (80px)

비율 기반:

  • • w-1/2 → 부모의 50%
  • • w-1/3 → 부모의 33.33%
  • • w-2/3 → 부모의 66.67%
  • • h-1/2 → 부모의 50%

전체 크기:

  • • w-full → 부모의 100%
  • • h-full → 부모의 100%

화면 기준:

  • • w-screen → 화면 너비 100%
  • • h-screen → 화면 높이 100%

임의 값:

  • • w-[200px] → 200px
  • • h-[50vh] → 화면 높이의 50%
  • • w-[300px] → 300px

최대/최소 크기:

  • • max-w-md → 최대 너비 28rem
  • • min-h-screen → 최소 높이 화면
  • • max-w-full → 최대 너비 100%

💡 예제 코드

<div class="w-40 h-20 bg-blue-200">박스</div>
<div class="w-[300px] bg-gray-200">고정 폭</div>
<div class="w-1/2 bg-gray-300">절반 폭</div>
