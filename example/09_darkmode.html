📖 Dark Mode 이론

Dark Mode는 dark: 접두사를 사용하여 다크 테마를 적용합니다.

기본 사용법:

  • • dark:bg-gray-900 → 다크 배경
  • • dark:text-gray-100 → 다크 텍스트
  • • dark:border-gray-600 → 다크 테두리

모드 전환 방법:

  • • <html class="dark"> → 다크 활성화
  • • <html> → 라이트 활성화
  • • JavaScript로 토글 가능

CDN 설정 (필수!):

  • • tailwind.config = {'{'}darkMode: "class"{'}'}
  • • CDN 위에 설정 스크립트 추가

Transition 추가:

  • • transition-colors → 부드러운 전환
  • • duration-300 → 0.3초

🎨 고급 다크모드 패턴

📖 고급 다크모드 이론

자주 사용하는 고급 다크모드 패턴입니다.

Ring (Focus 외곽선):

  • • dark:ring-blue-400 → 다크 ring 색상
  • • dark:ring-offset-gray-900 → ring offset

Divide (구분선):

  • • dark:divide-gray-700 → 다크 구분선
  • • 리스트 항목 구분에 유용

Placeholder:

  • • dark:placeholder-gray-500 → 입력 힌트
  • • input, textarea에 적용

Selection (드래그 선택):

  • • selection:bg-blue-500 → 선택 배경
  • • dark:selection:bg-blue-700
