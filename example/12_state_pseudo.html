🎭 State & Pseudo-class 개념

📖 State 이론

State(상태) 클래스는 요소가 특정 상황에 있을 때 적용되는 스타일입니다.

기본 상태:

  • • hover: → 마우스 올림
  • • focus: → 포커스 상태
  • • active: → 클릭 순간
  • • disabled: → 비활성화

요소 선택:

  • • first: → 첫 요소
  • • last: → 마지막 요소
  • • odd: → 홀수 요소
  • • even: → 짝수 요소

관계 선택:

  • • group-hover: → 부모 상태
  • • peer-focus: → 형제 상태

중요 팁:

  • • transition과 함께 사용
  • • 여러 상태 조합 가능
  • • 가장 많이 사용

🎯 Focus — 포커스 상태

📖 Focus 이론

Focus는 입력 요소가 선택되었을 때 적용됩니다.

주요 사용:

  • • focus:outline-none → 기본 외곽선 제거
  • • focus:ring-2 → ring 추가
  • • focus:border-blue-500 → 테두리 색상

Tip:

  • • outline 제거 + ring 추가
  • • 접근성을 위해 반드시 표시
  • • 브랜드 색상 사용

🔢 First / Last — 첫/마지막 요소

📖 First / Last 이론

First / Last는 리스트나 그리드의 첫 번째/마지막 요소를 선택합니다.

사용 방법:

  • • first: → 첫 번째
  • • last: → 마지막
  • • first:font-bold → 첫 요소 굵게

Tip:

  • • 리스트 구분선 제거
  • • 첫/마지막 요소 강조
  • • 테이블 스타일링

🔄 Odd / Even — 홀수/짝수

📖 Odd / Even 이론

Odd / Even은 홀수/짝수 번째 요소를 선택합니다. 테이블 줄무늬에 최적입니다.

사용 방법:

  • • odd: → 홀수 (1, 3, 5...)
  • • even: → 짝수 (2, 4, 6...)
  • • odd:bg-gray-100

Tip:

  • • 테이블 줄무늬 (Zebra Stripe)
  • • 리스트 가독성 향상
  • • 데이터 테이블

👥 Group — 부모 상태로 자식 제어

📖 Group 이론

Group은 부모 요소의 상태에 따라 자식 요소를 제어합니다.

사용 방법:

  • • 부모에 group 클래스
  • • 자식에 group-hover: 접두사
  • • group-hover:text-blue-500

Tip:

  • • 카드 hover 시 내부 텍스트 변경
  • • 이미지 hover 시 확대
  • • 버튼 hover 시 아이콘 이동

실습 17: Group 기본 사용

📝 실습 과제

  • • 부모에 group 클래스
  • • h3에 group-hover:text-blue-600
  • • p에 group-hover:text-gray-900

🔗 Peer — 형제 상태 감지

📖 Peer 이론

Peer는 형제 요소의 상태에 따라 다른 요소를 제어합니다. 주로 폼에서 사용합니다.

사용 방법:

  • • 기준 요소에 peer 클래스
  • • 대상 요소에 peer-focus:
  • • peer-checked:, peer-invalid:

Tip:

  • • 체크박스 선택 시 레이블 변경
  • • input focus 시 라벨 색상
  • • validation 에러 표시

실습 19: Peer Focus

📝 실습 과제

  • • input에 peer 클래스
  • • label에 peer-focus:text-blue-500
