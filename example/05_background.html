🎨 Background — 배경 설정

📖 Background 이론

Background는 요소의 배경색, 배경 이미지, 위치 등을 설정합니다.

배경색 (background-color):

  • • bg-gray-100 → 회색 배경
  • • bg-blue-500 → 파란 배경
  • • bg-[#ffddaa] → HEX 색상
  • • bg-blue-500/50 → 투명도 50%

배경 이미지:

  • • bg-[url('/img.png')] → 이미지
  • • bg-gradient-to-r → 그라데이션
  • • from-blue-400 → 시작 색
  • • to-purple-500 → 끝 색

배경 크기/위치:

  • • bg-cover → 전체 덮기
  • • bg-contain → 전체 보이기
  • • bg-center → 가운데 정렬
  • • bg-no-repeat → 반복 안 함

그라데이션 방향:

  • • bg-gradient-to-r → 좌→우
  • • bg-gradient-to-b → 위→아래
  • • bg-gradient-to-br → 대각선

🔲 Border — 테두리 설정

📖 Border 이론

Border는 요소의 테두리 두께, 색상, 스타일을 설정합니다.

테두리 두께 (border-width):

  • • border → 1px 테두리
  • • border-2 → 2px 테두리
  • • border-4 → 4px 테두리
  • • border-[3px] → 3px 임의 값

방향별 테두리:

  • • border-t-2 → 위쪽만 2px
  • • border-b-4 → 아래쪽만 4px
  • • border-l → 왼쪽만 1px
  • • border-r-2 → 오른쪽만 2px

테두리 색상 (border-color):

  • • border-gray-300 → 회색
  • • border-blue-500 → 파란색
  • • border-[#ff0000] → HEX

테두리 스타일 (border-style):

  • • border-solid → 실선
  • • border-dashed → 파선
  • • border-dotted → 점선
  • • border-double → 이중선

⭕ Radius — 모서리 둥글기

📖 Radius 이론

Radius는 요소의 모서리를 둥글게 만듭니다.

기본 둥글기:

  • • rounded → 약간 둥글게 (4px)
  • • rounded-sm → 조금 (2px)
  • • rounded-md → 보통 (6px)
  • • rounded-lg → 많이 (8px)
  • • rounded-xl → 아주 많이 (12px)

특수 둥글기:

  • • rounded-full → 완전 둥글게
  • • rounded-none → 각지게
  • • rounded-[20px] → 임의 값

방향별 둥글기:

  • • rounded-t-lg → 위쪽만
  • • rounded-b-lg → 아래쪽만
  • • rounded-l-lg → 왼쪽만
  • • rounded-r-lg → 오른쪽만

모서리별 둥글기:

  • • rounded-tl-lg → 왼쪽 위
  • • rounded-tr-lg → 오른쪽 위
  • • rounded-bl-lg → 왼쪽 아래
  • • rounded-br-lg → 오른쪽 아래

➗ Divide — 내부 구분선

📖 Divide 이론

Divide는 형제 요소들 사이에 자동으로 구분선을 만듭니다.

세로 구분선 (divide-y):

  • • divide-y → 세로 구분선 1px
  • • divide-y-2 → 2px
  • • divide-y-gray-300 → 회색

가로 구분선 (divide-x):

  • • divide-x → 가로 구분선 1px
  • • divide-x-2 → 2px
  • • divide-x-gray-400 → 회색

주의사항:

  • • 부모 요소에 적용
  • • 자식 요소들 사이에만 선 생성
  • • 첫 번째 자식 앞에는 선 없음

활용 예:

  • • 메뉴 리스트 구분
  • • 버튼 그룹 구분
  • • 목록 항목 구분
